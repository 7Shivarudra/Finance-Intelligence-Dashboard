# 💰 Finance Intelligence Dashboard

A premium financial analytics dashboard built using Streamlit.

## 🚀 Features

* 📊 Expense & income tracking
* 📈 Investment analytics (ROI)
* 🏦 Loan tracking & EMI analysis
* 🔄 Multi-file comparison
* 📅 Time-based filtering
* 🤖 Insight engine
* 📊 Correlation analysis

## 📂 Dataset

Synthetic datasets generated using a custom financial simulator.

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
